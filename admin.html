Editor de Trazado de Rutas

Esta página es para uso exclusivo del administrador.

Instrucciones:
  1. Selecciona una ruta del menú desplegable.
  2. Usa la herramienta de línea (✍️) a la izquierda del mapa para dibujar el trazado.
  3. Haz clic en el último punto para terminar. La ruta se guardará automáticamente.
  4. (Para editar una ruta existente, simplemente repite los pasos. Se borrará el trazado anterior).
